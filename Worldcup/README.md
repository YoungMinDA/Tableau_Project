#  월드컵 결과 분석(1930~2014)

### 1.개요

- 태블로의 기초를 다지기 위해 DataScienceRoadMap Youtube를 참고하여 제작한 대시보드
- 월드컵은 전 세계에서 가장 인기 있는 대회임

### 2.사용된 기술
- Tableau

### 3. 결과
- 프로젝트에서는 1930년부터 2014년까지의 월드컵 결과 데이터를 바탕으로 나라별로 나누어 인사이트를 도출
- 대시보드를 통해 각 나라의 우승, 준우승 및 연도를 쉽게 파악할 수 있습니다.
